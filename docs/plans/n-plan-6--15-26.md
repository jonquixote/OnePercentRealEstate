## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| DB pool saturation from MVT tile requests | Use pg_tileserv (separate container) not Next.js route |
| Worker OOM during MV refresh | Increase memory limit to 512M |
| Backfill fires triggers on 600k rows | Disable triggers with `SET session_replication_role` |
| Part 2 async migration applied too early | Gate on worker verification checklist |
| nginx config breaks on reload | Remove duplicate `map` block from proxy-params.conf |
| Tile cache miss storm after deploy | Add nginx proxy_cache with 60s TTL |
